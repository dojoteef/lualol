Module lol.matchlist

Allows for making queries against the League of Legends Matchlist API

By first creating an api object and creating a new matchlist object from it you can then make queries against the League of Legends Summoner API.

matchlist:validateFilters (filters)
Validates that the passed in filters for correctness

Remarks:

    This method may mutate the passed in filters to make them valid (such as swapping beginIndex and endIndex)

Parameters:

  • filters a table specifying how to filter the matchlist
    • championIds table an array-like table of champion IDs to use for fetching games.
    • seasons table an array-like table of seasons to use for fetching games,
    • rankedQueues table an array-like table of ranked queue types to use for fetching games,
    • beginTime long the begin time to use for fetching games specified as epoch milliseconds
    • endTime long the end time to use for fetching games specified as epoch milliseconds
    • beginIndex long the begin index to use for fetching games
    • endIndex long the end index to use for fetching games

Returns:

    a new matchlist object

matchlist:getBySummonerId (summonerId, opts)
Get a list of matches by Summoner id that statisfy the passed in filters

Parameters:

  • summonerId long the id of the Summoner whose matchlist you wish to retreive
  • opts table a table of optional parameters

    • expire: how long in seconds to cache a response (defaults to 1 hour, i.e. 60*60)
    • filters: a table specifying how to filter the matchlist
      • championIds: an array-like table of champion IDs to use for fetching games.
      • seasons: an array-like table of seasons to use for fetching games,
      • rankedQueues: an array-like table of ranked queue types to use for fetching games,
      • beginTime: the begin time to use for fetching games specified as epoch milliseconds
      • endTime: the end time to use for fetching games specified as epoch milliseconds
      • beginIndex: the begin index to use for fetching games
      • endIndex: the end index to use for fetching games
    • callback: a callback which receives the response from the API (data, code, headers)
