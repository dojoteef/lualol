Module lol.game

Allows for making queries against the League of Legends Game API

By first creating an api object and creating a new game object from it you can then make queries against the League of Legends Summoner API.

Class game

This class encapsulates manipulating the League of Legends Game API
game:game (api)
Create a new game object

Parameters:

  • api the api object that communicates with the League of Legends server

Returns:

    a new game object
game:getBySummonerId (summonerId, opts)
Get a list of recent games by Summoner id

Parameters:

  • summonerId long the id of the Summoner whose recent games you wish to retreive
  • opts a table with optional parameters:
    • expire long how long in seconds to cache a response (defaults to 5 hours, i.e. 5*60*60)
    • callback function a callback which receives the response from the API (data, code, headers)
