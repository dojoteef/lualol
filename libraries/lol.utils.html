Module lol.utils

Useful utility functions needed by the library

There are some useful utilities such as getting the epoch time that Lua doesn't natively support which are defined in this module.

Functions

Functions

epoch (time)
Lua doesn't have a portable way to get the epoch time (though for most systems os.time() is epoch time). This is a simple way to get the epoch time using just the Lua standard library functions that I believe should be portable.

Parameters:

  • time an optional time parameter to find the epoch time of, if not provided it defaults to using os.time() (NOTE: The passed in time must be one gotten from os.time() on the current system)
