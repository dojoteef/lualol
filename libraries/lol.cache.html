Module lol.cache

Basic caching for the League of Legends API

This module is one of the core modules that everything is built on top of.

If you want to create an api object, a cache object will be created for you automatically.

Functions

cache (cacheDir) Create a new cache object
_cache:clearAll () Clear all entries from the cache.
_cache:clearExpired () Clear all expired entries from the cache.
_cache:get (key) Get an entry from the cache if it exists and hasn't expired.
_cache:set (key, value, expireSecs) Add an entry to the cache with an optional number of seconds before expiration

Create a new cache object

Parameters:

  • cacheDir
    • the directory where to store cache entries

Returns:

_cache:clearAll ()
Clear all entries from the cache. This includes on disk and in memory.
_cache:clearExpired ()
Clear all expired entries from the cache. This includes on disk and in memory.

Get an entry from the cache if it exists and hasn't expired.

Parameters:

  • key
    • the key of the entry to find

Add an entry to the cache with an optional number of seconds before expiration

Parameters:

  • key
    • the key of the entry to store
  • value
    • the value of the entry to store
  • expireSecs
    • optional number of seconds before the entry expires
