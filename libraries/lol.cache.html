Module lol.cache

Basic caching for the League of Legends API

This module is one of the core modules that everything is built on top of.

If you want to create an api object, a cache object will be created for you automatically.

Class cache

This class encapsulates making adding and retreiving elements from a cache
cache:cache (cacheDir)
Create a new cache object

Parameters:

  • cacheDir string the directory where to store cache entries

Returns:

    a new cache object
cache:clearAll ()
Clear all entries from the cache. This includes on disk and in memory.
cache:clearExpired ()
Clear all expired entries from the cache. This includes on disk and in memory.
cache:get (key)
Get an entry from the cache if it exists and hasn't expired.

Parameters:

  • key the key of the entry to find (must be convertible to JSON)

Returns:

    the value that was previously stored for the given key
cache:set (key, value, expires)
Add an entry to the cache with an optional number of seconds before expiration

Parameters:

  • key the key of the entry to store (must be convertible to JSON)
  • value the value of the entry to store (must be convertible to JSON)
  • expires optional number of seconds before the entry expires
